# BEE - Event-Based Blob Frequency Estimator 
## Bachelor's Thesis Project 

in progress...

This projects implements event-based blob tracking using DBSCAN 
for detection and the SORT algorithm(Kalman filter + Hungarian matching) for tracking. 

Frequency estimation comming to repo soon...

## Dependancies:
- pandas
- opencv-python
- scikit-learn
- scipy
- PyYAML
- coded on Python 3.12.3

config file for configurations based on a custom dataset(data unavalaible publicly)
